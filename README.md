# Cats-n-Dogs Breed Detector

## Description

Cats-n-Dogs is a web app that accept any user-supplied image as input. If a dog or a cat is detected in the image, it will provide an estimate of the dog's or the cat's breed. If a human is detected, it will provide an estimate of the dog or cat breed that is most resembling.

## Key Features

* Keras using TensorFlow backend
* InceptionV3 used in deep CNN model via transfer learning
* OpenCV used in human face detection
